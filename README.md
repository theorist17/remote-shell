# remote-shell

This project implements remote shell.

## Geting Started

### Prerequites

Things you need to install this software

```
GNU Make
```
### Installing

A step by step series of commands to get a development env running

```
sudo apt-get install build-essential
make clean
make
```

## Running the tests

### Run server

```
./server
```

### Run client

```
./client
```
